# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern friendship management web application where users can track and maintain meaningful relationships. Users can log calls, texts, and video interactions with friends, monitor communication goals, and visualize interaction analytics through beautiful charts.

---

# ✨ Features

## ✅ Friend Management

- View all friends in a responsive card layout
- See friend status, tags, and days since contact
- Navigate to detailed friend profile pages

## ✅ Quick Check-In System

- Log Call, Text, and Video interactions
- Automatic timeline updates
- Real-time toast notifications

## ✅ Timeline Filtering

- Filter interaction history by:
  - Call
  - Text
  - Video

## ✅ Friendship Analytics

- Interactive Pie Chart using Recharts
- Visualize communication patterns
- Dynamic analytics updates

## ✅ Responsive Design

- Fully responsive for:
  - Mobile
  - Tablet
  - Desktop

## ✅ Additional Features

- Custom 404 Page
- Loading animations
- Dynamic date system
- Context API state management
